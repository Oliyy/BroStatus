# BroStatus

A fun project using web sockets to display and update friends status' who live in the same house but on crazy timezones

## Server
Recommend Node version: `16.13.2`

### How to run
`cd server`
`npm install`
`node src/index`


## Client

### How to run  
`cd client`
`npm install`
`npm start` or `npm run build`
